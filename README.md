# 🐍 Tkinter para Principiantes — Guía de Estudio

> Repositorio creado para estudiar interfaces gráficas con Python + Tkinter.
> Todo está explicado en español, con ejemplos listos para copiar y adaptar.

---

## ¿Cómo ejecutar un ejemplo?

```bash
python3 nombre_del_archivo.py
```

Asegúrate de tener Python instalado. Tkinter viene incluido con Python, no necesitas instalar nada extra.

---

## 📚 Contenido del repositorio

| Carpeta | ¿Qué aprendo? |
|---|---|
| `01_ventana_basica/` | Crear la ventana principal, darle título, tamaño y color |
| `02_widgets_texto_y_botones/` | Label, Button, Entry, Text — los ladrillos básicos |
| `03_layouts/` | Cómo organizar los widgets en la ventana (pack, grid, place) |
| `04_eventos_y_funciones/` | Conectar botones a funciones, leer lo que escribe el usuario |
| `05_widgets_avanzados/` | Checkbutton, Radiobutton, Listbox, Combobox |
| `06_ventanas_emergentes/` | Cuadros de diálogo y mensajes (messagebox) |
| `07_oop_con_tkinter/` | ⭐ Estructura OOP obligatoria en exámenes |
| `08_persistencia_datos/` | ⭐ Guardar datos en archivo (no se pierden al cerrar) |
| `09_patron_crud/` | ⭐ Agregar, Buscar, Editar y Eliminar registros |
| `10_mini_proyectos/` | ⭐⭐ Aplicaciones completas listas para adaptar |
| `CHEATSHEET.md` | Hoja de trampa rápida — repaso de último minuto |

> ⭐ = MUY importante para exámenes tipo parcial

---

## 🔥 Ruta rápida para el examen (estudia en este orden)

1. **`07_oop_con_tkinter/clase_ventana.py`** — La estructura que debes memorizar
2. **`08_persistencia_datos/guardar_json.py`** — Cómo guardar datos en archivo
3. **`09_patron_crud/crud_base.py`** — El patrón que se repite en todos los exámenes
4. **`10_mini_proyectos/sistema_biblioteca.py`** — El ejemplo más parecido al parcial real
5. **`CHEATSHEET.md`** — Repaso final

## 🗂️ Descripción de los mini proyectos

| Archivo | Descripción |
|---|---|
| `calculadora.py` | Calculadora básica con botones en grid |
| `lista_de_tareas.py` | To-do list con guardado automático en JSON |

| `inventario_tienda.py` | CRUD de productos con categorías y stock |
